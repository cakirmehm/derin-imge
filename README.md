# DERIN IMGE RUN

Derin İmge Run is an adventure game coded with vanilla JS for my lovely daughter.

Play on netlify: https://derin-imge.netlify.app/

## GUIDE

-   Arrow Up or touch for jumping.
-   SPACE or touch slide X for shooting.
